This short talk will explain how javascript handles being both single-threaded and non-blocking. We'll demystify the "Event Loop" and how concurrency works in javascript. If a coworker ever helped you solve a bug by wrapping a function with a <span style="font-family: Courier New, Courier, monospace;">setTimeout(myFunc, 0)</span> and you felt cheated, this talk is definitely for you.<br>
