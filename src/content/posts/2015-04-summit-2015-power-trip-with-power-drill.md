Forter has been using Riemann for almost a year in production to monitor our highly complex distributed system. We are using Riemann as our hub for alerts(Pager Duty), latency and exception reporting(Kibana) , continuous integration and system probes (GitHub). This presentation will cover Riemann patterns for maintenance-mode, state machine based alerts, statistical alerts, system and integration tests based alerts, event enrichment and aggregation for reporting, integration with GitHub status API. Learn how Riemann's unique architecture can scale linearly to handle high event throughput. Ohh yeah, there's going to be Clojure, lots of it.<br>
