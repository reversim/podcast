</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">באולפן האפל - רן תבורי, מנחה מאזן - אורי להב.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">הכול התחיל בעקבות הרצאה שנתן אחד, </span><a href="https://github.com/holman"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">זק הולמן</span></a><span style="vertical-align: baseline; white-space: pre-wrap;">, שמסביר איך </span><a href="http://zachholman.com/posts/how-github-works/"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">הדברים עובדים ב-Github</span></a><span style="vertical-align: baseline; white-space: pre-wrap;">. במקרה הוא גם מפתח בגיט האב.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">Github התחילה מבוזרת (על פני גיאוגרפיות, אזורי זמן וכו') והצורך לבנות טכנולוגיות תומכות הנחה את החברה מיומה הראשון.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">משרדים? זה פסה (אלא אם זה נוח לכם ואם צריך להיפגש עם מישהו בסביבה נורמלית).</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><a href="http://automattic.com/"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">יש</span></a><span style="vertical-align: baseline; white-space: pre-wrap;"> גם </span><a href="http://www.reversim.com/2012/03/127-gogobot.html"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">אחרים</span></a><span style="vertical-align: baseline; white-space: pre-wrap;"> שעובדים מבוזר.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">מי מכיר את </span><a href="https://github.com/github/hubot"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">האיש שבקיר (של חדר הצ'ט)</span></a><span style="vertical-align: baseline; white-space: pre-wrap;">?</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">שעות עבודה? גם זה פסה. באמנות כמו באמנות - יצירתיות לא באה לפי הזמנה.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">סביבת עבודה אופטימלית היא לא משהו לזלזל בו. במיוחד </span><a href="http://www.ted.com/talks/julian_treasure_the_4_ways_sound_affects_us.html"><span style="color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">אם היא רועשת</span></a><span style="vertical-align: baseline; white-space: pre-wrap;">.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
<b id="internal-source-marker_0.21075259777717292" style="font-weight: normal; text-align: start;"><span style="vertical-align: baseline; white-space: pre-wrap;">למיקרוסופט, כמו למיקרוסופט, יש הוראות איך לתקשר בשביל למזער את ההפרעה.</span></b></div>
</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">

</li>

</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">

</li>
<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">

<li style="font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt;">
