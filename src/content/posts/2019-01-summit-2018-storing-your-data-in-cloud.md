We all use cloud object storage extensively, like Amazon S3, Azure Blob Storage, Google Cloud Storage and others. However, not many utilize its unique capabilities and advanced features, such as versioning, security, life cycle... Throughout my work developing Ceph object storage and supporting large customers, I’ve seen a lot of incorrect and sub-optimal usage. This session will briefly discuss object storage design, highlight valuable features and review practical dos and don’ts.</div>
