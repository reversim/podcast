</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.21967930416576564" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">הכלי המשמש כ-Code Repo ואופן השימוש בו מכתיב אתן התקשורת בין אנשים.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>

</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>
