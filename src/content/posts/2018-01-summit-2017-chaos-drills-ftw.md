Chaos Drills can contribute a lot to your services resilience, and it’s actually quite a fun activity. We’ve (Outbrain) built a tool called GomJabbar to help you run those drills. GomJabbar is similar to Netflix's Chaos Monkey, but was built for a more relaxed environment / platform requirements, and allow it to run on your private cloud infrastructure.<br>
In this talk I'll explain why we built it, and how you can utilize it to improve your infrastructure and services resilience to failures.</div>
