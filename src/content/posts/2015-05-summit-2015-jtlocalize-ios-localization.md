JTLocalize is an open source internationalization &amp; localization framework for iOS, developed by JoyTunes.<br>
<br>
When we wanted to localize our app Piano Maestro, we discovered that there are several issues that are very hard to deal with using the tools Apple gave us for localizing iOS apps. For instance: effectively internationalizing Interface Builder documents, and handling localization of diffs between different versions of the app.<br>
<br>
We eventually developed our own infrastructure, that includes:<br>
<ul style="text-align: left;">
<li>An Objective-C Pod with some localization utilities &amp; localizable UI components for your iOS app.</li>
<li>A python package &amp; a command line tool that will handle the process of preparing string files for translation, handling diffs between different versions of the app, and more.</li>
</ul>
