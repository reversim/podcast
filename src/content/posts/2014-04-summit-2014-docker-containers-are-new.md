A quick talk about Docker, a totally awesome open source project which seems to generate a lot of buzz since its release in March 2013. Docker is a tool for packaging lightweight VMs called containers, which can be thought about as an "application in a box" that you can deploy wherever you want. In our talk we will see how Docker fits perfectly into a modern distributed, cloud-based, micro-service oriented architecture.</div>
