How to promote helpful code reviews by gamifying the code review process, using the open-sourced github-comment-counter: <a href="https://github.com/tzachz/github-comment-counter">https://github.com/tzachz/github-comment-counter</a>. Research shows that you can drive people to act by awarding them with points, no matter how meaningless these points are. At Kenshoo, we rank our reviewers and present the leaderboard for all to see, which incurred a ~130% bump in number of code review comments.<br>
