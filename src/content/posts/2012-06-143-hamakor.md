<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">במסגרת כנס האורניתולוגיה השישי הנערך בימים אלו בישראל, התכנסנו לפאנל מומחים בעניין שכל ציפור אינה יכולה בלעדיו - </span><a href="http://www.hamakor.org.il/"><span style="background-color: transparent; color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">המקור</span></a><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">נפרוש כנף עם רן תבורי ואורי להב, יגביה עוף יחד איתנו </span><a href="http://kaplanopensource.wordpress.com/"><span style="background-color: transparent; color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">ליאור קפלן</span></a><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">. ליאור חבר בוועד עמותת המקור, מתפרנס מקוד פתוח כבר כמה שנים (עושה אינטגרציה להפצות לינוקס) ו</span><a href="http://qa.debian.org/developer.php?login=kaplan%40debian.org"><span style="background-color: transparent; color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">חולה על דביאן</span></a><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;"></span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">עמותת המקור בישראל נוסדה בשנת 2003 (זה הכול התחיל בסרט) ומטרתה לקדם קוד פתוח בישראל.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">גוף רשמי יכול להיות מיוצג בקלות רבה הרבה יותר מול גופים רשמיים אחרים.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">כבר הזכרנו את </span><a href="http://august.penguin.org.il/"><span style="background-color: transparent; color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">אוגוסט פינגוין</span></a><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;"> ?</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">פעם בשנה העמותה בוחרת פרוייקט בקוד פתוח שזוכה בפרס. </span><a href="http://www.cs.technion.ac.il/~danken/cgi-bin/hspell.cgi"><span style="background-color: transparent; color: #1155cc; vertical-align: baseline; white-space: pre-wrap;">HSpell</span></a><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">, לדוגמה.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">
<b id="internal-source-marker_0.18284324114210904" style="font-weight: normal; text-align: -webkit-auto;"><span style="background-color: transparent; vertical-align: baseline; white-space: pre-wrap;">עמותות דומות בחו"ל מתחלקות לשני סוגים: עמותת גג ועמותה המוקדשת לפרויקט מסוים.</span></b></div>
</li>
<li style="background-color: transparent; font-family: Arial; font-size: 15px; list-style-type: disc; vertical-align: baseline;"><div style="margin-bottom: 0pt; margin-top: 0pt; text-align: right;">

</li>

</li>
